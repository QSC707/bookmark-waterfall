# 书签主页扩展 | Bookmark Homepage Extension

一个功能强大的 Chrome 新标签页扩展，将您的书签转化为优雅的多列瀑布流布局，提供高效的书签管理和访问体验。

![扩展预览](./screenshots/preview.png)

## ✨ 主要特性

### 📚 智能书签展示
- **多列瀑布流布局**：书签以优雅的多列形式展示，自适应屏幕宽度
- **层级导航**：点击文件夹即可展开下一级内容，支持无限层级
- **响应式设计**：从手机到4K显示器，完美适配各种屏幕尺寸

### 🎯 高效操作
- **多选支持**：Ctrl/Cmd + 点击多选，Shift + 点击范围选择
- **右键菜单**：完整的上下文菜单，支持打开、编辑、移动、删除等操作
- **拖拽排序**：直接拖拽书签和文件夹进行重新排序和移动
- **空格预览**：按住空格键悬停在文件夹上，快速预览文件夹内容而不展开，松开空格自动关闭预览
- **悬停展开**：可在设置中启用鼠标悬停自动展开功能，自定义延迟时间（默认500ms）

### 📅 最近添加书签
- **时间筛选**：按日期范围查看最近添加的书签
- **快速过滤**：一键查看今天、最近7天、最近30天的书签
- **排除规则**：自定义排除规则，过滤不想显示的书签

### 🎨 个性化设置
- **主题切换**：支持浅色/深色主题
- **悬停设置**：自定义悬停打开延迟时间
- **打开方式**：选择在当前标签页或新标签页打开书签
- **自定义背景**：设置个性化的横幅背景图片

### 🔍 其他功能
- **历史记录**：快速访问浏览历史
- **常用网站**：显示最常访问的网站
- **键盘快捷键**：支持常用操作的键盘快捷键
- **性能优化**：采用多项优化技术，确保流畅体验

## 🚀 安装方法

### 从源码安装

1. 克隆或下载本仓库：
```bash
git clone https://github.com/QSC707/Home2026.git
```

2. 打开 Chrome 浏览器，进入扩展管理页面：
   - 地址栏输入 `chrome://extensions/`
   - 或通过菜单：更多工具 → 扩展程序

3. 开启右上角的"开发者模式"

4. 点击"加载已解压的扩展程序"

5. 选择本项目的文件夹

6. 完成！打开新标签页即可使用

## 📖 使用指南

### 基本操作

**打开书签**
- 单击：在新标签页打开（可在设置中更改）
- Ctrl/Cmd + 单击：在新标签页打开
- Shift + 单击：在新窗口打开

**文件夹导航**
- 单击文件夹：展开下一级内容
- 悬停文件夹：自动预览内容（需在设置中启用）

**多选操作**
- Ctrl/Cmd + 单击：多选/取消选择
- Shift + 单击：范围选择
- 右键菜单：对选中项执行批量操作

**拖拽操作**
- 拖拽书签：移动到其他位置或文件夹
- 拖拽文件夹：重新排序或移动到其他文件夹

### 右键菜单功能

- 新标签打开 / 新窗口打开 / 隐身模式打开
- 重命名 / 修改网址
- 移动到其他文件夹
- 复制网址
- 查看属性
- 删除
- 新建文件夹
- 排序（按字母、日期、访问次数）

### 最近书签筛选

1. 点击右侧的星星图标打开侧边栏
2. 使用快速过滤按钮或自定义日期范围
3. 点击排除规则按钮设置过滤规则

## 🛠️ 技术特性

- **原生 JavaScript**：无框架依赖，轻量高效
- **Chrome Extension Manifest V3**：使用最新的扩展规范
- **性能优化**：
  - 事件委托减少监听器数量
  - DOM 缓存避免重复查询
  - 防抖节流优化高频操作
  - 虚拟滚动处理大量数据
- **安全性**：
  - XSS 防护
  - URL 协议验证
  - 安全的 DOM 操作

## 📋 权限说明

本扩展需要以下权限：

- `bookmarks`：读取和管理您的书签
- `history`：访问浏览历史记录
- `topSites`：显示常用网站
- `favicon`：获取网站图标
- `windows`：在新窗口打开链接
- `clipboardWrite`：复制网址到剪贴板
